Projects and other stuff I've done

TripLegal

TripLegal is a web based application that was designed for the TripAdvisor legal department, and is used to help manage litigation holds placed on employees. The backend API is written in Python and runs off of a Flask server. The application utilizes a PostgreSQL database, and the psycopg2 library to allow the Flask server to interact with the database. The front end is written in Vanilla JS with the fetch library being used for HTTP calls. I also used the Bulma CSS framework to assist with some of the design.

TickerDisplay for Reddit

TickerDisplay is a chrome extension designed to work with the old Reddit design. I frequently browse several financial-related subreddits and found that I was opening a large number of tabs to check on how the company a post mentioned was doing. The chrome extension uses a content script to go through the page and generates a small bubble whenever a post mentions a ticker symbol, it displays the current price as well as the percentage or dollar change for that day, as specified by the user. Uses Google's chrome extension API, as well as the IEX API to retrieve live stock prices directly from the exchange. The github page for that project can be found here

Easy Animator

Class project for an Object Oriented Design class, the Easy Animator was a java application that, when given a properly formatted text file, converted the text file to an animation that could be looped, played back at different speeds, and exported to an SVG animation. Utilized Java's swing library for UI design. The github page for that project can be found here
